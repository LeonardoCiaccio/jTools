Cos'è e a cosa serve questa libreria ?

Installer

Se non siete dei sviluppatori ignorate questa libreria, Installer serve a installare un bookamrklet o per meglio dire un bookmarklet che necessita di un framework quindi librerie, stili, etc..

Io personalmente lo utilizzo per progetti importanti, ma vediamo come utilizzarlo : 

                                

                                    installer.install( {
                                        
                                        createFrame : false,
                                        where       : document.body,
                                        onProgress  : function(){},
                                        onFinish    : function(){},
                                        elements    : [
                                            
                                            "http://miosito.com/mialib.js",
                                            "http://miosito.com/miostile.css",
                                            "http://miosito.com/miocontroller.php?installer=js"
                                            
                                        ]

                                    } );

Come possiamo vedere nell'esempio precedente per installare script e css basta inserire i link direttamente nell'array 'elements' verrà riconosciuto automaticamente, in caso di risorse dinamiche come ad esempio su file 'php' basterà aggiungere alla fine il parametro aggiuntivo 'installer=js' oppure 'installer=css', importante ricordare che 'createFrame' settato a 'true' crea un div dove verranno installati gli elementi di fatto bypassa 'where' per il resto nulla d'aggiungere visto che i parametri parlano da se.

Per disinstallere versioni precedenti : 

                                

                                    installer.uninstall( callback );

La funzione 'callback' verrà eseguita al termine della disinstallazione. Le altre 2 funzioni ovvero 'installer.getSignature()' e 'installer.getFrameID()' sono utili per lavorare con il 'div' creato.
